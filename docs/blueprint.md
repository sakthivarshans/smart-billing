# **App Name**: SmartBill

## Core Features:

- RFID Item Scan: Simulate scanning RFID items to add them to the current bill, using a mock data generator.
- Dynamic Table Update: Dynamically update the dashboard table with item name, price, and date/time upon simulated RFID scan.
- Total Amount Calculation: Automatically calculate and display the total bill amount as items are scanned.
- WhatsApp Number Input: Allow the customer to enter their WhatsApp number for sending the receipt.
- Payment Redirection: Redirect to the payment page upon clicking 'Proceed to Payment'.
- Payment Simulation: Simulate a successful payment on the payment page and return to the dashboard.
- WhatsApp Receipt: Send an automatic WhatsApp receipt to the entered number after successful payment (placeholder function for now).

## Style Guidelines:

- Primary color: Deep Indigo (#4B0082) to evoke professionalism and trust.
- Background color: Light Lavender (#E6E6FA), a desaturated version of indigo, creating a calming yet professional atmosphere.
- Accent color: Purple (#800080), analogous to deep indigo, used for buttons and highlights.
- Body and headline font: 'Inter', a sans-serif font, providing a modern and neutral look, suitable for both headlines and body text.
- Use a clean and modern layout with a focus on readability and ease of use.
- Use simple, clear icons to represent different actions and elements.
- Use subtle animations and transitions to enhance the user experience and provide feedback on interactions.
